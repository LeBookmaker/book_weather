# book_weather
Change the weather with chat commands

## Chat commands
```/weather [Weather Type]```

## Weather Types
- Blizzard
- Clouds
- Drizzle
- Fog
- GroundBlizzard
- Hail
- HighPressure
- Hurricane
- Misty
- Overcast
- OvercastDark
- Rain
- Sandstorm
- Shower
- Sleet
- Snow
- SnowClearing
- SnowLight
- Sunny
- Thunder
- Thunderstorm
- WhiteOut

## How to install
* Copy and paste ```book_weather``` folder to ```resources/book_weather```
* Add ```ensure book_weather``` to your ```server.cfg``` file
* Enjoy!
